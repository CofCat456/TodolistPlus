*   新增任務
*   查看任務列表
*   更新任務狀態
*   刪除任務

(讓使用者能直接回到主頁面)
*   漢堡選單
(保持畫面整潔，讓使用者可以依照使用狀況開關)
*   添加代辦事項開關
(保持畫面整潔，讓使用者可以依照使用狀況開關)
*   新增時間小工具
(讓使用者可以了解現在時間，方便查看自己規劃進度)
